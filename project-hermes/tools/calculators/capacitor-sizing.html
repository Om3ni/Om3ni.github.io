Measure amps through one capacitor lead and voltage across the terminals while the unit is running. Constant 2652 = 10⁶ / (2π × 60). Use 3183 at 50 Hz.
How to measure
Capacitors hold charge with power off. Discharge with an insulated 20 kΩ / 5 W resistor across the terminals and verify 0 V before handling.
  1. Identify terminals: HERM (compressor), FAN (condenser fan), C (common, to contactor).
  2. Set clamp meter to AC amps. Clamp around only one lead on the terminal you're testing (HERM for compressor section, FAN for fan section).
  3. Restore power, run in cooling, let it stabilize 30–60 seconds.
  4. Read and enter the amperage above.
  5. Switch meter to AC volts. Measure tested-terminal to C. Enter that voltage above.
  6. Enter the printed nameplate MFD. Within ±6% is good; 6–10% is marginal; beyond 10% replace the cap.

How to pick a value
  1. Read motor HP from the nameplate or body label. If missing, estimate from the equipment spec (condenser fan motors are usually 1/6–1/2 HP).
  2. Read line voltage at the motor leads with the unit off. Common values: 115, 208, 230, 277, 460 V.
  3. Pick the nearest standard size to the midpoint: 3, 4, 5, 7.5, 10, 12.5, 15, 17.5, 20, 25, 30, 35, 40, 45, 50, 55, 60, 70, 80 µF.
  4. Voltage rating: 370 VAC for 208/230 V systems, 440 VAC for 277 V, 660 VAC for 460/480 V, 250 VAC for 115 V. Higher rating is always fine and extends life.
  5. After installing, run the unit and switch to Verify mode. Motor should not run hot, hum, or draw excessive current.
Last-resort estimate. Undersized caps reduce torque and overheat the motor; oversized caps overdrive the start winding. Prefer the manufacturer's specification in every case.
Verify formula derives from I = V · 2πfC (capacitor reactive current). Size rule-of-thumb: typical 12–18 µF/HP at 230 V for PSC run capacitors, scaled by 1/V² for the same reactive support at other voltages.
