Situation Assessment
Contractor Lockout

When a prior contractor has password-protected the CU 352 and withheld credentials on contract transition, setpoints, firmware version, sensor configuration, and cascade logic are all hidden. Work through the access paths below before considering a controller replacement.

Check Before You Assume

Both operator and service passwords ship disabled from the factory. If the prior contractor never enabled them, the panel may be open. Try navigating directly to Settings — if it prompts for a password, proceed to the service code below.

What Remains Readable Without Authentication

Status display (Menu 1) and Alarm Log (Menu 3.2) are accessible without a password. Use these to gather visible system information — active alarms, current operating mode, process values — before proceeding.

Override Access
Service Code Override

The CU 352 includes a universal service-level override for exactly this situation — bypasses a contractor-set operator password without wiping any configuration.

CU 352 Universal Service Code

When prompted for a password at the panel keypad, enter:

6814
After You're In

Photograph every accessible screen before making any changes. Document setpoints, sensor configuration, operating mode, and cascade settings. Even a locked panel may contain tuned parameters worth preserving. Set a new password under credentials your organization controls before leaving the site.

Software
PC Tool Setup

Grundfos PC Tool E-products is the commissioning and service software for all CU 352 / Hydro MPC systems. Freeware, Windows XP–11. Install and update before leaving for the site.

01
Download the Installer
Get PC Tool E-products from the Grundfos Resources page. Run MSIApp.exe — handles the Visual C++ runtime dependency automatically.
02
Install & Launch
Accept defaults. Launch via Start → Programs → Grundfos PC Tools → PC Tool E-products. Skip COM port selection until the CU 352 is physically connected.
03
Update GSC Library Do This Before You Leave
Go to Tools → Update configuration files. Pulls current GSC file library from Grundfos servers. Required before standard configuration files can be pushed. Needs internet access.
04
Connect to CU 352
Power up the Control MPC panel. Connect via USB-B cable. The Network List populates with icons for CU 352 and any IO 351 modules detected on the GENIbus network.

Read Before You Write

When connecting to an existing unlocked controller, record all current parameters before pushing any GSC files. Sending a standard configuration overwrites existing custom settings.

NPSHa-Sensitive Installations

On systems with marginal suction conditions — long suction runs, shallow tanks, or known cavitation history — establish speed limits and conservative ramp rates before bringing pumps online. Aggressive cascade startup on a compromised suction side causes immediate mechanical seal damage.

Commissioning
GSC File Load Order

Configuration is delivered as a numbered sequence of GSC files via PC Tool Standard Configuration. Sequence is fixed. Do not restart the CU 352 between files.

One Restart — After All Files Are Sent

Restart button is in the lower-right corner of PC Tool. Clicking mid-sequence initializes on incomplete configuration. Send all required files first, then restart once.

1
Control MPC Always Required
Standard Configuration → Application → 1. Control MPC. Select system type and pump count. Search, select file, click Send. Verify GSC number appears in label position 1.
2
Control MPC Options If Applicable
Required only if IO 351B interface modules (A01/A03) are installed. Skip if none present.
3
Hydro MPC Always Required
Select 3. Hydro MPC. Encodes outlet pressure range, sensor range, and dry-running protection type. Match sensor range to transducer on discharge manifold.
4
Hydro MPC Options If Applicable
Required only if redundant primary sensor is installed. Skip if single-sensor configuration.
5
Pump Data Always Required
Select 5. Pump data. Search by pump model string or GSC file number from nameplate label. Loads hydraulic performance curve for installed pump.
  • If pilot pump installed, load its GSC file after this step
  • Pump data must also be sent to each E-pump individually via step-by-step E-pump configuration
6
Restart CU 352
After all files are sent, click Restart in lower-right corner of PC Tool. Initialization takes approximately 25 seconds.
Commissioning
Pressure Setpoint

Verify against a static head calculation from actual installation geometry. Do not carry over a setpoint without confirming the math.

Static Head Calculation

Height above pump (ft) × 0.433 = static head PSI. Add minimum fixture pressure (15–20 PSI at highest fixture) plus estimated friction losses. Start 2–4 PSI above calculated minimum. Verify top-floor pressure before locking in.

ComponentFormulaNotes
Static HeadHeight (ft) × 0.433Measure above pump discharge — not above grade.
Fixture Pressure+15 to +20 PSIUse 20 PSI for care facility applications.
Friction Losses+3 to +8 PSI typicalHigher on aged cast iron distribution.
Minimum SetpointSum of aboveProgram 2–4 PSI above minimum during startup. Walk down after top-floor verification.
Sensor Tap Location Changes the Number

Sensor at pump manifold — setpoint must account for all head and friction to highest fixture. Sensor downstream — setpoint is lower. Confirm sensor location before programming.

Commissioning
Cascade & Dry-Run

Default cascade logic is appropriate for most installations — adjust only if startup observation warrants it.

01
Lead/Lag & Alternation
Default uses automatic alternation to equalize run hours. Leave at default unless a specific lead pump is required for diagnostic purposes.
02
Cascade Staging Tune on Suction-Sensitive Systems
On suction-sensitive installations, aggressive staging creates transient low-pressure events before the system stabilizes.
  • If NPSHa margin is tight, limit max simultaneous pump starts to 1
  • Increase minimum sequence time if cavitation noise is observed on startup
03
Dry-Run Protection Configure Before First Run
Configure inlet pressure or level switch input before bringing pumps online. Without this, a suction-side failure runs pumps dry.
  • Alarm 206 — Warning threshold (pump continues, alert issued)
  • Alarm 214 — Alarm threshold (all pumps stopped, manual reset)
  • A nuisance stop is recoverable — a dry-run failure is not
04
Speed Limiting
If NPSHa analysis has identified a safe max speed, set per-pump limits via Custom Configuration. The E-pump VFD will not exceed this regardless of what cascade logic requests.
Commissioning
E-Pump GENIbus Address

Each E-pump must be assigned a unique GENIbus address before the CU 352 can communicate with it. Use Grundfos GO app (Bluetooth) or GO Remote (IR). Pumps are numbered left to right as installed.

Address Assignment

Point the Grundfos GO at the IR window on the pump terminal box. Navigate to Settings → NUMBER, set the address, press OK. Work one pump at a time — shield others if within IR range to prevent address conflicts.
